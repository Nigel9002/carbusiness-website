1. admin.html → uploadToImageKit(file)
2. → waitForImageKit() (✅ SDK loads)
3. → fetch('/imagekit-auth')  ← THIS IS WHERE IT FAILS
4. → imagekit-auth-server.js or Firebase Function
5. → Returns auth data (token, expire, signature, publicKey, urlEndpoint)
6. → new window.ImageKit({ ... })
7. → imagekit.upload()
